# Matthias Silberhain - Author Website

## Projektübersicht

Eine moderne, responsive Autoren-Website mit dunklem Design, Admin-Dashboard und umfassenden Verwaltungsfunktionen.

## Features

### Hauptwebsite
- Dunkles Design mit Silber-Akzenten
- Responsive Layout (Mobile First)
- Preloader mit Logo und Typewriter-Effekt
- Dark Mode Toggle (Schwarz/Dunkelgrau)
- Bücher-Präsentation mit Leseproben und Kauf-Links
- Veranstaltungskalender
- Kontaktbereich

### Admin-Dashboard
- Geschützter Login-Bereich
- Bücherverwaltung (CRUD)
- Cover-Upload (PNG/JPG)
- PDF-Leseproben Upload
- Kauf-Links Verwaltung
- Veranstaltungsverwaltung
- Social Media Profile
- Design-Einstellungen (Farben, Hintergründe)
- Allgemeine Einstellungen

### Technische Features
- PHP Backend API
- MySQL Datenbank
- Sichere Authentifizierung
- Datei-Uploads mit Validierung
- Logging-System
- CORS-Unterstützung
- Responsive Admin-Panel

## Installation

### Voraussetzungen
- PHP 8.0 oder höher
- MySQL 5.7 oder höher
- Apache oder Nginx
- Node.js (für optionales Build-Tooling)
